covid_news_handling module

This module retrieves news articles from a news api and performs functions such as removing articles from a list, returning the list and scheduling, the list to be updated.

covid_news_handling.article_compare(article_list_param: list) → list

checks whether an article title is in the removed article_title_list and does not append that article to the article list if the case is true

Parameters

article_list (list) –

A list containing dictionarys of articles :returns: :rtype: A list containing dictionarys of articles

covid_news_handling.news_API_request(covid_terms: str = 'Covid COVID-19 coronavirus') → list

Makes a request using the request module to newsapi.org where it retrieves a list of articles containing the covid_terms

Parameters

A string value containing terms that are passed into the request query to filter the returned articles :returns: ARTICLE_LIST :rtype: list
